Privacy Policy

Last Updated: 2025-11-14

Summary

We believe your documents belong to you. Owl.md is an offline-first application.

  • No personal data collection
  • No user accounts required
  • No tracking or analytics
  • Files stay on your device

Information We Do NOT Collect

Owl.md does NOT collect, store, or transmit:

  • Personal information (name, email, etc.)
  • User accounts or credentials
  • Location data
  • Usage statistics or analytics

Data Stored Locally

The following data stays on your device and is never sent to our servers:

1. Document Content

Your Markdown and PDF files are stored in your device's local storage.

2. App Preferences

Theme selection, PDF settings, and history are stored effectively using Android SharedPreferences.

Network Requests

The app only connects to the internet for:

  • GitHub Import (Optional): If you choose to load a file from a URL.
  • Announcements: Checking for app updates via a public JSON file on GitHub.
  • External Images: Loading images you add via URL.

Third-Party Services

Owl.md does NOT use:

  • Analytics services (No Firebase, No Google Analytics)
  • Crash reporting tools
  • Advertising networks
