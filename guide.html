üéØ Advanced Features

Automatic Table of Contents

Add the marker [TOC] anywhere in your document to auto-generate a clickable Table of Contents with page numbers.

Title Page Metadata

Add YAML frontmatter to the very top of your file to generate a professional cover page:

---
title: My Annual Report
author: John Doe
date: 2025-12-14
version: 1.0
---

üîó GitHub Integration

You can load Markdown files directly from any public repository.

  1. Tap the Folder Icon.
  2. Select Load from GitHub URL.
  3. Paste the link (blob or raw format).

‚ùì Troubleshooting

PDF Won't Generate?

  • Check if your Markdown syntax is valid.
  • Ensure you have storage space.
  • Try switching themes.

Images Not Loading?

  • For network images, ensure you have internet.
  • For local images, ensure the file wasn't deleted.
  • Check the URL format (must be HTTPS).
