Powerful Features

Everything you need to create professional PDFs

Live Preview with Smooth Animations

See your changes instantly as you type with professional slide animations. Real-time markdown rendering with split view support on desktop and tablets.

  • ✓ Instant markdown parsing with 60 FPS rendering
  • ✓ Split view on desktop/tablets (768px+)
  • ✓ Smooth slide animations between editor and preview
  • ✓ Syntax highlighting and callout blocks
  • ✓ Tables, lists, and code block support

4 Professional Themes

Choose from 4 carefully crafted themes, each with unique fonts and color palettes. Professional and Minimalist feature varied heading weights for typography excellence.

  • ✓ Professional - Inter font with light H1 headers
  • ✓ Modern - Asap font with vibrant orange accents
  • ✓ Minimalist - Ubuntu font, black/gray only
  • ✓ Classic - Noto Serif with traditional ivory background

Images & Advanced Formatting

Insert images from device or URLs with custom sizing and alignment. Add metadata, table of contents, and clickable links that work in PDFs.

  • ✓ Image insertion with width/alignment control
  • ✓ Clickable links in generated PDFs
  • ✓ Auto table of contents with [TOC] marker
  • ✓ Document metadata & beautiful title pages
  • ✓ 18-button formatting toolbar with undo/redo

BiDi Support & Smart File Management

Full RTL support for Arabic and other right-to-left languages. Load files from device, GitHub, or recently opened history.

  • ✓ Complete Arabic/RTL language support
  • ✓ Load from GitHub URLs (public repos)
  • ✓ Recent files history (last 20 files)
  • ✓ Share PDFs to any app instantly
  • ✓ Works 100% offline (except GitHub/URLs)
